# Freelance Admin Time Calculator

Interactive calculator from the 2026 Smallpdf Freelancer Freedom Index,
a survey of 397 freelancers, consultants, and solo operators.

Calculate how many hours and dollars per year you're losing to admin work
based on your typical monthly volume of proposals, contracts, invoices,
and client onboardings.

Hosted via GitHub Pages. Survey conducted in April 2026 by Smallpdf
in partnership with Fractl.
